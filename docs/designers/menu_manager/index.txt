[h1]Menus[/h1]

The Menu Manager in the control panel provides a simple, web-based interface for managing any number of navigation menus on your web site.  After building your menu in the control panel, it can be dynamically included in any template with the [span_code]{menu}[/span_code] template plugin described below.

Designers can use multiple CSS "class" and "id" attribute options to style the automatically-generated menu.

[h2]Assigning CSS Classes to Individual Menu Items[/h2]

[APPNAME] allows you to control each menu item's [span_code]<li>[/span_code] element's "class" attribute from within the Menu Manager.  Simply load your menu for editing, find the link you want to edit, select "Edit", and enter at least one class name into the "CSS Class" box.  If you want to add multiple classes, separate them with a space.  Then, when your menu items are retrieved with [span_code]{menu}[/span_code], this item will appear as:

[code]
<li class="your_class">
	<a href="#">link text</a>
</li>
[/code]

[h2]Template Plugins[/h2]

[h3]menu (parameters: name, show_sub_menus)[/h3]

Displays a menu in formatted HTML.  Menus exist as one [span_code]<ul>[/span_code] with any possible sub-menus as [span_code]<ul class="children">[/span_code] elements within their parent link's [span_code]<li>[/span_code] element.  The active menu element is labelled with the "active" class.

Required parameters:

* [span_code]name[/span_code] - The menu name.  It is selected when you add a menu and can be seen in the left menu of the Menu Manager.

Optional parameters:

* [span_code]show_sub_menus[/span_code] - Enter either "yes", "no", or "active".  If "active", only the sub-menu of the active menu item will be shown.  "yes" shows all sub menus and "no" shows none.
* [span_code]show_sub

Usage:

[code]
<div class="my_menu">
	{menu load="main_menu" show_sub_menus="active" class="my_menu_ul" id="my_menu_id"}
</div>
[/code]

Returns:

[code]
<div class="my_menu">
	<ul class="my_menu_ul" id="my_menu_id">
		<li class="active link_1_class">
			<a href="#">link 1</a>
			<ul class="children">
				<li>
					<a href="#">sub menu link 1</a>
				</li>
				<li class="active">
					<a href="#">sub menu link 2</a>
				</li>
			</ul>
		</li>
		<li class="another_class">
			<a href="#">link 2</a>
		</li>
		...
	</ul>
</div>
[/code]
